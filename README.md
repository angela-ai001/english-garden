# english-garden

國小英語單字配對遊戲網站。

## GitHub Pages 發布方式

1. 將此 repo 推到 GitHub（預設分支為 `work`）。
2. 到 GitHub 專案頁面：
   - `Settings` → `Pages`
   - `Build and deployment` 選擇 **GitHub Actions**。
3. Push 後會自動觸發 `.github/workflows/deploy-pages.yml`。
4. 部署完成後，網址格式通常是：
   - `https://<你的 GitHub 帳號>.github.io/english-garden/`

> 若 repo 名稱不同，請把網址中的 `english-garden` 改成你的 repo 名稱。
